# Prepora Website

A modern, responsive marketing website for the Prepora meal planning app.

## Overview

This is a single-page website showcasing the Prepora app's features, benefits, and screenshots. Built with vanilla HTML, CSS, and JavaScript for optimal performance and easy hosting.

## Features

- **Responsive Design**: Works perfectly on desktop, tablet, and mobile devices
- **Modern UI**: Matches the app's calming aesthetic with coral/peach and sage green colors
- **Smooth Animations**: Subtle scroll animations and transitions
- **Fast Loading**: No heavy frameworks, pure HTML/CSS/JS
- **SEO Optimized**: Proper meta tags and semantic HTML

## File Structure

```
/app/website/
├── index.html      # Main HTML file
├── styles.css      # All styling
├── script.js       # Interactive elements
└── README.md       # This file
```

## Sections

1. **Navigation**: Fixed header with smooth scroll links
2. **Hero**: Eye-catching headline with app tagline
3. **Features**: 6 key features of the app
4. **How It Works**: 3-step process explanation
5. **Screenshots**: 4 mockup previews of the app
6. **Benefits**: 4 key benefits for users
7. **CTA**: Download section with app store buttons
8. **Footer**: Links and company information

## Deployment

This website can be hosted on:
- **Netlify**: Drag and drop the `/app/website` folder
- **Vercel**: Connect to Git repository
- **GitHub Pages**: Push to repository and enable Pages
- **Any static hosting**: Upload files via FTP

## Customization

### Colors
Edit the CSS variables in `styles.css`:
```css
:root {
    --primary-color: #FF9B85;
    --secondary-color: #A8C5A8;
    --accent-color: #B8A8D8;
    ...
}
```

### Content
Edit text directly in `index.html`

### Links
Update the app preview link in the CTA section (currently points to the Expo preview)
